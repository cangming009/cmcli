cmcli

CMC's Unified CLI — search, extract, bilibili in one tool

MIT License v0.1.0 Python ≥ 3.12 typer + rich
# Install in 10 seconds 
pip install -e ~/Desktop/claude/cli/cmcli
# Search, extract, automate — three commands 
cmcli search "AI Agent" --engine bing,baidu --lang zh-CN
cmcli extract https://www.bilibili.com/video/BVxxxx
cmcli bilibili user-videos 18052876 --limit 5
🔍

Search (SearXNG)

244 engines. Privacy-friendly metasearch via local docker instance. No API keys needed.

📦

Extract (url-tools)

Pull subtitles from B站 / YouTube videos. Plain text, SRT, or JSON output.

📺

Bilibili (opencli)

Monitor UP主, track hot videos, search B站 — all from terminal.

🚀 Quick Install

cd ~/Desktop/claude/cli/cmcli
pip install -e .

Prerequisites

# SearXNG (for search)
docker run -d --name searxng-core -p 8080:8080 searxng/searxng:latest

# url-tools (for extract)
cd ~/Desktop/claude/cli/url-tools-cli && pip install -e .

# opencli (for bilibili)
npm install -g @jackwener/opencli

Commands

cmcli search "query" [flags]
# Basic search (default: bing + baidu)
cmcli search "Claude Code MCP"

# Chinese search, time filter, markdown output
cmcli search "大模型" --lang zh-CN --time month --md

# Specific engines
cmcli search "opencli" --engine google,github,arxiv

# JSON for scripting
cmcli search "Hermes Agent" --json
cmcli extract <url> [flags]
# Extract B站 video subtitle
cmcli extract https://www.bilibili.com/video/BV1TZ5Y6CEH1

# Output as SRT
cmcli extract <url> --format srt -o subtitle.srt

# JSON output
cmcli extract <url> --json
cmcli bilibili <subcommand> [flags]
# User's recent videos
cmcli bilibili user-videos 18052876 --limit 5

# Search B站
cmcli bilibili search "AI编程" --limit 10

# Hot videos
cmcli bilibili hot --limit 20

Options — cmcli search

FlagDefaultDescription
--engine, -ebing,baiduComma-separated engines (google,baidu,bing,github,arxiv,wikipedia...)
--lang, -lautoLanguage code (auto, zh-CN, en, ja...)
--time—Time range: day, week, month, year
--limit, -n10Max number of results
--jsonfalseOutput raw JSON
--mdfalseOutput as Markdown table
--urllocalhost:8080SearXNG instance URL
